あなたは日にちでは何歳ですか？計算するのは簡単です - 今日からあなたの誕生した日を引くだけです。 私達はそれにもかかわらずこれを実際の課題にして日付の差分を数えます。

あなたは年、月、日の3つの数字のタプルとして2つの日付が与えられます。 例えば1982年4月19日は (1982, 4, 19) です。あなたは与えられた日付の差を探す必要があります。 例えば今日と明日の差は1日です。差は常に正の数字か0なので絶対値を取るのを忘れないで下さい。

入力 2つの日付が整数のタプルとして与えられる

出力 2つの日付の差を整数として返します

例 

days_diff((1982, 4, 19), (1982, 4, 22)) == 3
days_diff((2014, 1, 1), (2014, 8, 27)) == 238
days_diff((2014, 8, 27), (2014, 1, 1)) == 238

どのように使われるか Pythonはバッテリーが内蔵されています、このミッションでは完全なモジュールの使い方を学ぶ必要があり、車輪の再発明をする必要はありません。

事前条件 日付は1年1月1日から9999年12月31日の間です。 日付は正しいです。
